# URL Shortener

A simple web application to shorten long URLs instantly.  
Built with **HTML, JavaScript, Netlify Functions, and Supabase**.

## Live Demo
[Your Netlify URL here]

## Features
- Paste a long URL and get a short URL
- Redirects from short URL to original
- Tracks number of clicks
- Copy-to-clipboard functionality

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Netlify Functions
- Database: Supabase
